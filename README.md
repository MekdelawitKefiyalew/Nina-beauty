# 💜 Beauty Salon Website

## Overview

This project is a front-end website designed for a beauty salon, created to provide an elegant and user-friendly
experience for clients who want to explore services and make bookings. The design focuses on a soft, feminine aesthetic
with a calming color palette, aiming to reflect the atmosphere of a real salon.

## Features

* ✨ Clean and modern homepage
* 💇‍♀️ Service selection interface before booking
* 📝 Booking form with a soft and welcoming design
* 📖 About Us page to introduce the salon
* 📞 Contact page for easy communication
* ⭐ Review section for customer feedback

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose of the Project

This project was built as a learning experience and a way to practice front-end development skills.
It reflects my growing understanding of web design, layout structuring, and user experience.
